Built a dashboard to analyze booking trends, cancellation patterns, and operational KPIs using SQL, Excel, and Power BI.

- Power BI
- Excel

## KPIs Analyzed
- Cancellation Rate
- Booking Trends
- Customer Behavior
- Driver Performance

## Key Insights

- Analyzed peak booking periods
- Improved operational visibility using dashboards
